jQuery管理包装集

确定包装集的大小

  • 使用length属性
  • 使用size()方法

从包装集中获取元素

  • 使用索引，例如$(‘img’)[0]
  • 使用get()方法，例如$(‘img’).get(0)

    get()方法接受负值的下标作为参数，get(-1)返回包装集中的最后一个元素

  • get()方法不指定参数，则返回元素数组，但更推荐使用toArray()方法
  • 使用索引、get()、toArray()方法都是返回DOM元素，如果想返回包含一个元素的包装集，可以使用下面的方法：
    • eq(index)
    • first()
    • last()
    • $($(‘p’).get(0))

  • 可以使用indox(element)获取下标，参数element可以是元素的引用，也可以是元素选择器。如果省略参数则返回第一个元素在同级节点中的下标，找不到则返回-1。

      // 获取id为findMe的图片，在整个页面图片集合中的下标
  var n = $('img').index($('img#findMe')[0]);

  // 获取第一个<img>元素在其父节点里的下标
  var nn = $('img').index();

    分解元素包装集

  • 添加集合

    • add(expression, context)
    • 返回包装集副本，并向其中添加由expression指定的元素
    • 参数expression可以是选择器、元素、数组，还可以是包含HTML标记的字符串，来添加新建的元素
    • 参数context可以是选择器、元素、数组、jQuery

  • 删除集合

    • not(expression)
    • 返回包装集的副本，从中删除匹配的元素。
    • 参数expression可以是选择器、元素、数组、函数
    • 如果传递的是函数，会为包装集中的每个元素调用此函数，this指当前元素，并删除返回值为true的元素
    • filter(expression)
    • 创建包装集的副本，并从中删除与expression不匹配的元素集合
    • 参数expression可以是选择器、元素、数组、函数

  • 获取包装集的子集

    • slice(begin, end)
    • 创建并返回新包装集，其中包含匹配集中的连续部分
    • 参数begin、end，为元素的开始和结束位置数字
    • has(test)
    • 创建并返回新包装集，此包装集只包含原始包装集中，子节点匹配test表达式的元素

  • 转换包装集中的元素

    • map(callback)
    • 为包装集中的每个元素调用回调函数，并将返回值收集到jQuery对象实例中

  • 遍历包装集中的元素

    • each(iterator)
    • 遍历匹配集里所有的元素，为每个元素调用传入的迭代函数
    • 该方法也可以遍历数组对象

  • 使用关系获取包装集

    • 往下选
      • children([selector])返回由每个包装元素所有的子节点（不包含重复的子节点）组成的包装集。

        children仅在直接子节点中查找元素而类似的find()方法是在所有后代节点中查找元素。

    • 往上选
      • offsetParent()返回由包装集中离第一个元素最近的，使用相对定位或者绝对定位的祖先元素组成的包装集。

        （offsetLeft,offsetTop）就是以这个祖先元素为参考点的

      • closest(selector)从祖先元素中选择最近的一个匹配元素
      • parent([selector])返回由每个包装元素的直接父元素（不包含重复元素）组成的包装集
      • parents([selector])返回由每个包装元素所有的祖先元素（不包含重复元素，不包含根节点）组成的包装集。
      • parentsUntil([selector])返回由每个包装元素所有的祖先元素组成的包装集，直至遇到选择器匹配的元素，但不包括此元素。
    • 往后选
      • next([selector])返回由每个包装元素后面下一个同级元素（不包含重复元素）组成的包装集。
      • nextAll([selector])返回由每个包装元素后面素有的同级元素组成的包装集
      • nextUntil([selector])返回由每个包装元素后面所有的同级元素组成的包装集，直至遇到与选择器相匹配的元素，但不包括次元素。
    • 往前选
      • prev([selector])返回匹配元素集合中每个元素紧邻的前一个同胞元素，通过选择器进行筛选是可选的。
      • prevAll([selector])返回由每个包装元素前面所有的同级元素组成的包装集
      • prevUntil([selector])返回由每个包装元素前面所有的同级元素组成的包装集，直至遇到选择器匹配的元素，但不包括此元素。
    • 平行选
      • siblings([selector])返回由每个包装元素的所有同级元素（不包含重复元素）组成的包装集
    • 其他
      • contents()返回由每个元素的内容组成的包装集，包括文本节点。

        该方法常用来获取<iframe>元素的内容

      • find(selector)在后代元素中查找匹配选择器的所有元素
      • is(selector)检查包装集中是否存在与选择器匹配的元素，返回布尔值。

        这是一个高度优化的快速操作，可用在性能优化领域中

管理jQuery链

  • end()获取方法链中，上一个返回的包装集。
  • andSelf()获取方法链中，最后的两个包装集合并后的包装集。
