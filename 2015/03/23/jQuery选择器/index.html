jQuery选择器

$()函数最常用的功能

  • 使用选择器选择DOM元素
  • 创建新的DOM元素

控制上下文

  • $()函数第一个参数是选择器，第二个参数是上下文。
  • 默认把DOM树的所有元素作为该选择器的上下文。
  • 上下文参数可以是DOM元素的引用，也可以是包含jQuery选择器的字符串，也可以是DOM元素包装集。
  • 调用某个$()函数的结果传递给另一个$()，即：
    $($(".hide")).hide();
  • 将选择器的应用范围限制在DOM树指定部分：
    $(selector,'div#sampleDOM')

使用CSS选择器

  • jQuery不依赖CSS在浏览器的实现，即使浏览器没有正确的实现标准CSS选择器，jQuery也会遵循W3C标准的规则来正确的选择元素。
  • jQuery允许使用逗号操作符将多个选择器合并成一个选择器。

使用子节点、容器和特性选择器

  • * 匹配所有元素
  • E 匹配标签名称为E的所有元素
  • E F 匹配标签名称为F的所有元素，这些元素是E的子节点
  • E>F 匹配标签名称为F的所有元素，这些元素是E的直接子节点
  • E+F 匹配标签名称为F的所有元素，这些元素是E后面的第一个兄弟节点
  • E~F 匹配标签名称为F的所有元素，这些元素是E后面的兄弟节点之一
  • E.C 匹配标签名称为E的所有元素，这些元素拥有CSS类名为C，如果省略E，则相当于*.C
  • E#I 匹配标签名称为E的所有元素，这些元素的ID特性值为I，如果省略E则相当于*#I
  • E[A] 匹配标签名称为E的所有元素，这些元素拥有特性A
  • E[A=V] 匹配标签名称为E的所有元素，这些元素的A特性值为V
  • E[A^=V] 匹配标签名称为E的所有元素，这些元素的A特性值以V开头
  • E[A!=V] 匹配标签名称为E的所有元素，这些元素的A特性值不等于V，或者不存在A特性
  • E[A*=V] 匹配标签名称为E的所有元素，这些元素的A特性值包含V

通过位置选择元素

  • 使用冒号开头，例如p:even
  • CSS规范中称为伪类，jQuery中称为过滤器，因为这些选择器是用来过滤另一个基础选择器的。
  • :first 匹配上下文中的第一个元素
  • :last 匹配上下文中的最后一个元素
  • :first-child 匹配上下文中第一个子节点
  • :last-child 匹配上下文中最后最后一个子节点

    :first与’:first-child’的区别：？？？

  • :only-child 返回所有没有兄弟节点的元素
  • :nth-child(n) 返回上下文中第n个子节点
  • nth-child( odd | even ) 匹配上下文中的奇数或偶数子节点
  • nth-child(Xn+Y) 匹配上下文中的由提供的公式计算出的子节点。如果Y是0，则可以省略。li:nth-child(3n)返回每个列表中能被3整除的列表项，而nth-child(5n+1)返回每个列表中所有能被5整除的列表项的下一个列表项

    :nth-child过滤器从1开始计数（为了与CSS兼容）

  • :odd 匹配上下文中的奇数元素
  • :even 匹配上下文中的偶数元素
  • :eq(n) 匹配第n个元素
  • :lt(n) 匹配第n个元素之前的元素（不包含第n个元素）
  • :gt(n) 匹配第n个元素之后的元素（不包含第n个元素）

使用CSS和自定义的jQuery过滤选择器

  • :animated 选择处于动画状态的元素
  • :button 选择按钮元素
  • :checkbox 选择复选框元素
  • :checked 选择处于选中状态的复选框或单选按钮
  • :contains('x') 选择包含文本x的元素
  • :disabled 选择处于禁用状态的元素
  • :enabled 选择处于启用状态的元素
  • :file 选择文件输入元素
  • :has(selector) 选择至少包含一个匹配指定选择器的元素的元素
  • :header 选择标题元素
  • :hidden 选择隐藏元素
  • :image 选择图片输入元素
  • :input 选择表单元素
  • :not(select) 选择不匹配指定选择器的元素
  • :parent 选择有子节点的元素，空元素除外
  • :password 选择口令元素
  • :radio 选择单选框元素
  • :reset 选择重置按钮元素
  • :selected 选择列表中处于选中状态的
  • :submit 选择提交按钮元素
  • :text 选择文本输入框元素
  • :visable 选择可见的元素

创建新元素

  • 以下三种写法是等价的，但建议使用符合标准的标记
    $("<div>");
$("<div></div>");
$("</div>");
  • 可以传递第二个参数到创建元素的$()方法中，用来指定特性和特性值，这个参数是一个JavaScript对象，他的属性作为特性的名称，相应的属性值作为特性值。例如：
    $('<img>',
  {
      src: 'img/aaa.jpg',
      alt: '产品图片',
      click: function(){
          alert($(this).attr('alt'));
      },
      css: {
          cursor: 'pointer',
          border: '1px solid black'
      }
  }
)
.appendTo('body');
